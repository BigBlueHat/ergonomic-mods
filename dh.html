DH Mods

DH mods, or Curl Mods, are specific to the Colemak layout. The aim is to make typing more comfortable by addressing some specific criticisms that have been made of the default Colemak layout. Specifically, the commonly needed D and H keys are moved away from the centre columns of the keyboard, to more readily accessible index-finger keys.

An important principle is that any key moved should still be typed with the same finger as default Colemak, to avoid any adverse effect on bigrams (i.e. letter pairs). Use of the Angle Mod is a prerequisite for applying a DH mod.

See the Colemak Mod-DH pages for a more full description.
